# 🔐 Secure Password Generator

A secure and customizable Password Generator built using Python.  
This project generates strong, cryptographically secure passwords based on user-selected preferences.

It uses Python’s built-in `secrets` module to ensure high security, making it suitable for real-world password generation.

---

## 🚀 Features

✅ Custom password length  
✅ Option to include:
- Uppercase letters (A–Z)
- Lowercase letters (a–z)
- Numbers (0–9)
- Special symbols (!@#$%^&*)  
✅ Cryptographically secure randomness (`secrets` module)  
✅ Interactive command-line interface  
✅ Lightweight and beginner-friendly  

---

## 🛠️ Technologies Used

- Python 3
- secrets module (for secure random generation)
- string module (for character sets)
